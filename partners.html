VR/AR

  • Virtual Reality

    • Требуется создать VR приложение для любой прикладной области. VR шлемы и контроллеры будут выданы участникам на мероприятии.

      Приз: Oculus Rift + Oculus Touch.

  • Augmented Reality

    • Требуется создать AR приложение для любой прикладной области с использованием контроллеров LeapMotion, которые будут выданы участникам на мероприятии.

      Приз: Leap Motion.

IoT

  • Samsung

    • Команде будет необходимо построить модульную систему жизнеобеспечения в космосе. Участникам будет предоставлено необходимое оборудование: типовой комплект Samsung IoT: модули на STM32L1 и радиоприёмопередатчике LoRa, всевозможные датчики, компоненты и периферия. Поверьте, у Вас будут разбегаться глаза.

      Приз: секретный приз от компании Samsung.

Социальные технологии

  • Aquarius

    • Командам необходимо создать решение, которое позволит сократить затраты системного администратора на решение рутинных задач.

  • Foresight

    • Задание на тему Machine Learning. Участникам будет необходимо разработать решение для тэгирования видеоконтента по заранее заданным объектам.

  • Сибинтек

    • Участникам необходимо разработать ИТ решение для доверительного управления финансами с использованием технологии blockchain.

В данном направлении будут выдаваться ценные призы от каждой компании.
